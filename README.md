## 3D Design Software

### Free 3D Design Software

#### Comprehensive 3D Suites
* **[Blender](https://www.blender.org/)** (offline) - Industry-standard open-source 3D creation suite for modeling, sculpting, animation, rendering, compositing, motion tracking, and game creation. Completely free with no limitations.

#### Voxel & Block-Based Modeling
* **[MagicaVoxel](https://ephtracy.github.io/)** (offline) - Lightweight yet powerful voxel art editor and path tracing renderer. Perfect for creating voxel-based 3D models and scenes with an intuitive interface.
* **[MagicaCSG](https://ephtracy.github.io/index.html?page=magicacsg)** (offline) - CSG (Constructive Solid Geometry) based voxel modeling tool from the creators of MagicaVoxel.

#### Browser-Based Modeling
* **[SculptGL](https://stephaneginier.com/sculptgl/)** (online) - Powerful browser-based 3D sculpting application with dynamic topology, multiresolution sculpting, and PBR rendering.
* **[Tinkercad](https://www.tinkercad.com/)** (online) - User-friendly browser-based CAD tool perfect for beginners, education, and quick prototyping. Great for 3D printing preparation.
* **[Clara.io](https://clara.io/)** (online) - Full-featured cloud-based 3D modeling, animation, and rendering platform with collaborative features.

#### Polygon Modeling
* **[Wings 3D](http://www.wings3d.com/)** (offline) - Open source subdivision modeler inspired by Nendo and Mirai. Excellent for low to medium poly modeling with an intuitive interface.
* **[SketchUp Free](https://www.sketchup.com/plans-and-pricing/sketchup-free)** (online) - Web-based version of the popular architectural and design modeling software. Great for architectural visualization and quick concept modeling.

#### Limited Free Tiers
* **[Vectary (Free Plan)](https://www.vectary.com/)** (online) - Browser-based 3D and AR design tool with a limited free plan. Features parametric modeling and real-time rendering.
* **[Womp3D (Free Tier)](https://womp.com/)** (online) - Real-time browser-based 3D modeling with innovative tools and some free features. Great for organic and creative modeling.

### Paid 3D Design Software

#### Professional 3D Suites
* **[Cinema 4D](https://www.maxon.net/cinema-4d)** (offline) - Professional 3D modeling, animation, simulation, and rendering software known for its ease of use and motion graphics capabilities.
* **[Houdini](https://www.sidefx.com/)** (offline) - Node-based procedural 3D software renowned for its powerful VFX, simulation, and procedural modeling capabilities.
* **[Modo](https://www.foundry.com/products/modo)** (offline) - Powerful and flexible 3D modeling, texturing, and rendering software with excellent polygon and subdivision surface modeling tools.

#### Digital Sculpting
* **[ZBrush](https://www.maxon.net/zbrush)** (offline) - Industry-standard digital sculpting tool for creating high-resolution models for movies, games, and digital art.
* **[3DCoat](https://3dcoat.com/)** (offline) - Complete 3D sculpting, retopology, UV mapping, and texturing solution with voxel and surface sculpting capabilities.
* **[Substance 3D Modeler](https://www.adobe.com/products/substance3d-modeler.html)** (offline) - VR-capable sculpting and modeling software by Adobe with intuitive desktop and VR workflows.
* **[Nomad Sculpt](https://nomadsculpt.com/)** (mobile) - Professional mobile 3D sculpting application for tablets with desktop-quality features.

#### Browser-Based Premium Tools
* **[Spline](https://spline.design/)** (online) - Real-time 3D design tool for creating interactive web experiences with animation and interactivity tools.
* **[Womp3D (Pro)](https://womp.com/pricing)** (online) - Full version with advanced features for professional browser-based 3D modeling.
* **[Vectary (Premium)](https://www.vectary.com/pricing/)** (online) - Paid plans unlock powerful browser-based 3D modeling, AR features, and collaboration tools.

## File Formats & Standards

### Common 3D File Formats
* **.blend** - Blender native format
* **.fbx** - Autodesk interchange format
* **.obj** - Wavefront object format (widely supported)
* **.gltf/.glb** - Modern web-friendly format
* **.usd/.usdz** - Universal Scene Description by Pixar
* **.stl** - Standard for 3D printing
* **.dae** - COLLADA interchange format
