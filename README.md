Covers 3D modeling, animation, VFX, rendering, game development, motion graphics, and digital art.

## Browse the catalog

Full catalog lives on the [interactive site](https://3d.devanshutak.xyz) — search, filter, and graph view. This README is a curated landing page; below is a hand-picked sample from each section.

### Assets & Libraries

Models, textures, HDRIs, audio, stock — everything downloadable.

- [Poly Haven](https://polyhaven.com/) — CC0 models, textures up to 8K+, HDRIs up to 16K.
- [Quixel Megascans + Bridge](https://quixel.com/) <span class="lic-pill lic-freemium">Freemium</span> — Scanned materials and import plugin. Free with Unreal Engine.
- [Sketchfab](https://sketchfab.com/) <span class="lic-pill lic-freemium">Freemium</span> — Massive downloadable library.
- [2D Pixel Art Character Template Asset Pack](https://zegley.itch.io/2d-platformermetroidvania-asset-pack) <span class="lic-pill lic-paid">Paid</span> — 2D character Art Assets to start your game off the right way. Perfect for sidescrollers, platformers, or metroidvania.
- [2D Pixel Dungeon Asset Pack](https://pixel-poem.itch.io/dungeon-assetpuck) — 16x16 pixel character and tileset

→ **[Browse all 243 entries in Assets & Libraries →](https://3d.devanshutak.xyz/sections/assets-libraries/)**

### Modeling, Sculpting & Texturing

Per-DCC learning plus texturing and material authoring toolchain.

- [Blender to Unity FBX](https://github.com/EdyJ/blender-to-unity-fbx-exporter) — Unity export addon.
- [Houdini Donut in Any Shape](https://vimeo.com/492505292) — Procedural donut by Lewis Orton.
- [Agama Materials](https://agama.itch.io/agama-materials) <span class="lic-pill lic-paid">Paid</span> — Software for creating PBR materials and painting 3d models. You can create materials and use them for painting models, or you can use pre-made textures.
- [ArmorPaint](https://armorpaint.org/) — Open-source texture painting.
- [AwesomeBump](https://github.com/kmkolasinski/AwesomeBump) — Generate normal, height, specular or ambient occlusion textures from a single image.

→ **[Browse all 263 entries in Modeling, Sculpting & Texturing →](https://3d.devanshutak.xyz/sections/modeling-sculpting-texturing/)**

### Animation & Rigging

Technical animation, rigging, motion capture.

- [Mixamo](https://www.mixamo.com/) — Rigged characters, auto-rigger, and free animation library (Adobe).
- [Rokoko Vision](https://www.rokoko.com/) <span class="lic-pill lic-freemium">Freemium</span> — Free AI mocap from webcam or video.
- [Cascadeur](https://cascadeur.com/) <span class="lic-pill lic-freemium">Freemium</span> — Physics-based AI animation with AutoPosing. Unreal plugin.
- [DeepMotion](https://www.deepmotion.com/) <span class="lic-pill lic-freemium">Freemium</span> — Game-ready character animation from video. BVH/FBX export.
- [Free Mo Cap](https://freemocap.org/) — Free open-source motion capture system (traditional, not AI).

→ **[Browse all 21 entries in Animation & Rigging →](https://3d.devanshutak.xyz/sections/animation-rigging/)**

### Lighting, Rendering & Shaders

3D lighting tutorials, rendering theory, shader theory, renderer-specific learning.

- [Creative Product Lighting with Redshift & C4D](https://www.youtube.com/@MaxonVFX) — Jarred van de Voort, NAB 2023.
- [USD Arnold](https://github.com/LumaPictures/usd-arnold) — USD Schemas and tools for exchanging Arnold shader information between multiple 3rd party packages.
- [Foundry Katana](https://www.foundry.com/products/katana) <span class="lic-pill lic-paid">Paid</span> — Look-development and lighting orchestration tool for film/episodic VFX, scaling to massive scenes via deferred evaluation.
- [Houdini](https://www.sidefx.com/products/houdini/) <span class="lic-pill lic-paid">Paid</span> — Procedural node-based DCC for VFX, simulation, and FX.
- [JanusVR](https://janusvr.com/) — Webpages as collaborative 3D webspaces interconnected by portals.

→ **[Browse all 245 entries in Lighting, Rendering & Shaders →](https://3d.devanshutak.xyz/sections/lighting-rendering-shaders/)**

- [Vizrt](https://www.vizrt.com/) <span class="lic-pill lic-paid">Paid</span> — Broadcast graphics.

Motion graphics tools and inspiration, video editing.

- [Friction](https://friction.graphics/) — Free open-source motion graphics and 2D vector animation. After Effects-style workflow.

### Art, Design & Visual Storytelling

Concept art, composition, color, typography, photography, cinematography, fundamentals.

- [Adobe Photoshop](https://www.adobe.com/products/photoshop.html) <span class="lic-pill lic-paid">Paid</span> — Industry-standard raster image editor and compositor; available on iPad.
- [Procreate](https://procreate.com/procreate) <span class="lic-pill lic-paid">Paid</span> — iPad-only digital painting app; one-time purchase, no subscription.
- [Krita](https://krita.org/) — Free open-source digital painting app from KDE; v5.3 and v6.0 shipped March 2026.
- [Adobe Express](https://new.express.adobe.com/) <span class="lic-pill lic-freemium">Freemium</span> — Quick Adobe design tool.
- [Adobe Illustrator](https://www.adobe.com/products/illustrator.html) <span class="lic-pill lic-paid">Paid</span> — Industry-standard vector graphics editor; available on iPad.

Generative AI and ML tools across image, video, 3D, texture, audio.

- [ComfyUI Registry](https://comfyui.org/) — Community workflow and subgraph sharing.

- [Runway Gen-4](https://runwayml.com/) <span class="lic-pill lic-freemium">Freemium</span> — Character consistency across shots, controllable camera.
- [3DTopia](https://github.com/3DTopia/3DTopia) — Open-source text-to-3D pipeline (coarse → refined).
- [ACE Studio](https://www.acestudio.ai/) <span class="lic-pill lic-freemium">Freemium</span> — AI singing voice synthesizer with expression control.

- [RizomUV](https://www.rizom-lab.com/) <span class="lic-pill lic-paid">Paid</span> — Fast UV unwrapping, auto-seam, auto-pack. Standalone or bridge.

- [3DF Zephyr](https://www.3dflow.net/3df-zephyr-photogrammetry-software/) <span class="lic-pill lic-freemium">Freemium</span> — Free lite version (up to 50 photos).
